Калькулятор (Windows Forms)
=================================

Краткое описание
-----------------
Простое WinForms-приложение — калькулятор, реализующий базовые арифметические операции, последовательные вычисления, % и backspace. Есть возможность сохранить / загрузить результат в текстовый файл и переключать цветовую тему (светлая/тёмная).

Требования
----------
- Visual Studio (рекомендуется) 2019/2022/2026
- .NET Framework 4.7.2

Как запустить
------------
1. Откройте решение Calculator(NETFramework).slnx в Visual Studio.
2. Постройте и запустите проект (F5).

Примечания
---------
- Сохранение/загрузка хранит одно числовое значение в текстовом файле.
- Рекомендуется вручную проверить привязки событий в дизайнере, если вносились правки.
